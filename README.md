## Flujo de Trabajo

```
main ──────────────────────────────────────► producción estable
  └── develop ────────────────────────────► integración y desarrollo
```

Cambios en `develop` → PR con aprobación del supervisor de par → merge a `main`.

```
feat(proceso): agregar DOP línea 3 PET
fix(scada): corregir comunicación OPC-UA
docs(vsm): actualizar VSM estado futuro línea 2
```

**Gestión del repositorio:** Samuel David Sanchez Cardenas es el responsable de la estructura, documentación transversal, MES Power BI y revisión de NX/Digital Factory.
