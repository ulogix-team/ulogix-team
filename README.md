# ulogix-team

Repositorio principal del equipo **Ulogix** — Proyecto Integrador APM 2026-1, Universidad Nacional de Colombia.  
Línea automatizada de producción de bebidas **FEMSA / Coca-Cola**: tres productos diferenciados, arquitectura ISA-95, PLC, robótica ABB, gemelo digital NX, SCADA Ignition y MES Power BI.

## Flujo de Trabajo

```
main ──────────────────────────────────────► producción estable
  └── develop ────────────────────────────► integración y desarrollo
```

Cambios en `develop` → PR con aprobación del compañero supervisor → merge a `main`.

**Convención de commits:**
```
feat(plc): agregar lógica Grafcet producto A
fix(scada): corregir comunicación OPC
docs(proceso): actualizar VSM línea 2
```

**Gestión del repositorio:** Samuel David Sanchez Cardenas es el responsable de mantener la estructura, documentación transversal y revisión de los repositorios del proyecto.
