# Actas de Reunión

Registro oficial de todas las reuniones del equipo Ulogix durante el semestre 2026-1.  
Las actas se suben en **formato PDF**.

<img src="https://raw.githubusercontent.com/ulogix-team/assets/main/dividers/divider-section-dark.svg" width="100%"/>

## Convención de nombres

```
YYYY-MM-DD_descripcion-breve.pdf
```

Ejemplos:
- `2026-03-03_primera-reunion.pdf`
- `2026-03-22_reunion-primera-entrega.pdf`

## Plantilla de acta

Cada acta debe incluir:

| Campo | Contenido |
|---|---|
| Fecha y hora | Fecha completa, hora de inicio y fin |
| Modalidad | Presencial / Virtual (plataforma) |
| Asistentes | Nombre completo de cada integrante |
| Ausentes | Nombre y justificación |
| Temas tratados | Listado de puntos discutidos |
| Decisiones tomadas | Acuerdos concretos del equipo |
| Compromisos | Responsable + tarea + fecha límite |
| Próxima reunión | Fecha tentativa |
