# Matriz RACI — Responsabilidades por Módulo

<img src="https://raw.githubusercontent.com/ulogix-team/assets/main/dividers/divider-section-dark.svg" width="100%"/>

Distribución formal de roles para cada área del proyecto integrador APM 2026-1 — **Propuesta Automatización FEMSA Fontibón**.

## Leyenda

| Letra | Rol | Descripción |
|:---:|---|---|
| **R** | **Responsible** | Ejecuta y entrega el trabajo |
| **A** | **Accountable** | Dueño final — aprueba y rinde cuentas |
| **C** | **Consulted** | Aporta criterio técnico antes de ejecutar |
| **I** | **Informed** | Recibe actualizaciones del avance |
| **S** | **Supervisor de par** | Revisa el trabajo del compañero asignado |

<img src="https://raw.githubusercontent.com/ulogix-team/assets/main/dividers/divider-section-dark.svg" width="100%"/>
